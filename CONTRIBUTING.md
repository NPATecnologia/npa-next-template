# Contribuindo

Obrigado por contribuir com um projeto NPA Tecnologia.

## Fluxo
1. Crie uma branch a partir de `main`: `feat/...`, `fix/...`, `chore/...`, `docs/...`.
2. Commits em [Conventional Commits](https://www.conventionalcommits.org/).
3. Abra um PR preenchendo o template. Squash merge.

## Padrões
- TypeScript strict, sem `any`.
- Datas de negócio em UTC-3.
- Sem segredos no código (`.env.local` é gitignored).
- LGPD: páginas legais quando houver coleta de dados.
