# Política de Segurança

## Reportando vulnerabilidades

Encontrou uma vulnerabilidade? Não abra issue pública — envie por email para **nathan@npatecnologia.com.br** com o assunto `[SECURITY] <descrição curta>`.

Resposta inicial: até 48h úteis.

## Escopo

Valem report de:
- Execução remota de código
- Injeção (SQL, XSS, command)
- Auth bypass / privilege escalation
- Exposição de secrets ou PII
- SSRF, path traversal, desserialização insegura

## Fora de escopo

- DoS volumétrico
- Issues em dependências já reportadas upstream
- Social engineering, phishing
- Clickjacking em páginas sem ação crítica
